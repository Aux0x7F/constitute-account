# constitute-account

`constitute-account` is the browser-side identity and runtime authority for
first-party Constitution app surfaces.

It handles account/session state, paired-device identity, grants, retained
runtime materialization, app launch context, product shell posture, runtime
diagnostics, and cross-app repair of browser state. Other browser apps use it as
the place where user authority, local runtime continuity, and prepared
capability posture meet.
